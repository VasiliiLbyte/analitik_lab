# Knowledge Base — Аналитик.Лаб

Структура базы знаний для AI-агентов.

## Содержимое

```
knowledge/
├── proposal_template.docx       # Jinja2-шаблон для генерации КП (используется сейчас)
├── examples/
│   ├── kp/                      # 8 реальных PDF коммерческих предложений (few-shot примеры)
│   └── requests/                # Заявки на отбор проб (DOCX + PDF)
├── samples/
│   └── application_template.docx  # Образец заявки
└── prices/                      # Прейскурант услуг (будет загружен позже)
```

## Как используется

### Фаза 1 (MVP) — сейчас
- `proposal_template.docx` — шаблон, в который Proposal Agent подставляет данные через docxtpl

### Фаза 2 (RAG) — планируется
- `examples/kp/` — few-shot примеры для Proposal Agent (улучшение качества КП)
- `examples/requests/` — примеры заявок для обучения Intake Agent
- `prices/` — прейскурант → Qdrant RAG для точного расчёта стоимости
- `samples/` — шаблоны документов для генерации

## Безопасность
Файлы в этой папке могут содержать коммерческую информацию.
НЕ добавляйте сюда персональные данные клиентов.
